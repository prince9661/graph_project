# Graph Algorithms Visualizer

An interactive web-based project to visualize and understand different **graph algorithms**.  
This project helps students and developers learn how graph algorithms work step-by-step through visual animations.

---

##  Features

- Visualize popular **graph algorithms**
- Step-by-step execution
- Interactive graph creation
- Add nodes and edges dynamically
- Weighted and unweighted graphs
- Algorithm animation for better understanding
- Clean and responsive UI

---

##  Algorithms Implemented

- Breadth First Search (BFS)
- Depth First Search (DFS)
- Dijkstra's Shortest Path Algorithm
- A* Pathfinding Algorithm
- Minimum Spanning Tree (Prim's Algorithm)
- Kruskal's Algorithm
- Topological Sorting
